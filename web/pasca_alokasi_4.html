Langkah-langkah untuk mengajukan permohonan izin Pasca Alokasi Lahan adalah sebagai berikut:

1.Login 

Pastikan Anda sudah memiliki untuk masuk ke dalam aplikasi portal BSW.

Jika belum memiliki, Anda dapat melakukan registrasi sebagai user portal melalui menu Register.

Jika sudah memiliki sebagai user portal, Anda dapat langsung login ke dalam portal.

2.Registrasi Perijinan Pasca Alokasi Lahan

Masuk ke halaman utama perijinan Pasca Alokasi Lahan dengan memilih menu Home > Layanan Perijinan > BP Batam > Pasca Alokasi Lahan.

Pada halaman perijinan Pasca Alokasi Lahan berisi 3 menu utama yaitu:

a.Panduan

Pada halaman ini berisi informasi umum mengenai perizinan Pasca Alokasi Lahan, tata cara dan persyaratan pengajuan izin, dan langkah-langkah untuk mengajukan perizinan.

b.Registrasi

Halaman registrasi berisi form yang harus diisi oleh user untuk merekam data permohonan izin. Sebelum masuk ke form pengisian data, user harus memilih jenis perizinan Pasca Alokasi Lahan yang dituju.

Selanjutnya, sistem akan mengarah ke halaman form registrasi. Halaman form registrasi terdiri dari 4 tahapan yaitu:

 1)Entry Data Diri

Mengisi informasi data pemohon, mengupload dokumen lampiran sesuai dengan tipe pemohoan yang diajukan dan menyetujui surat pernyataan untuk mendapatkan hak akses aplikasi perizinan Pasca Alokasi Lahan.

2)Registrasi Perijinan

Mengisi informasi data perizinan yang diajukan dan mengupload dokumen lampiran.

3)Konfirmasi Kedatangan

Memilih tanggal kedatangan di loket untuk proses verifikasi berkas.

4)Bukti Tanda Terima

Mendownload bukti tanda registrasi perizinan Pasca Alokasi Lahan. Bukti ini dapat digunakan sebagai tanda bukti saat proses verifikasi berkas di loket.

c.Monitoring

Halaman monitoring berisi daftar data perizinan yang pernah diajukan oleh user. Pada halaman ini, user dapat melihat detil data perizinan, memperbarui data, mendownload bukti tanda terima, dan memantau progress pengajuan izin.
